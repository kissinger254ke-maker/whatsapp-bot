# WhatsApp Bot Documentation

## Overview
This project is a WhatsApp bot that automates various tasks via the WhatsApp messaging platform.

## Features
- Automated messaging
- Response handling
- Task scheduling
- User interaction
- Integration with external APIs

## Installation
To get started with the WhatsApp bot, follow these instructions:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/kissinger254ke-maker/whatsapp-bot.git
   cd whatsapp-bot
   ```
2. **Install dependencies:**
   ```bash
   npm install
   ```
3. **Configure environment variables:**
   Create a `.env` file and set up the required variables:
   ```
   WHATSAPP_API_KEY=your_api_key
   ```

## Usage
To run the bot:
```bash
npm start
```
